開始客製化

關於這個範例

內含簡易的排版、響應式格線與一個右側側欄。適合作為個人作品頁或簡易介紹頁的起點。

如何客製化

  • 在 <head> 中修改 <title> 與描述。
  • 替換文字、加入圖片（放在 repo 的 images 資料夾）。
  • 要增加更多頁面，可建立其他 HTML 檔案並在 nav 加入連結。

小技巧

  • 要做使用者網站：儲存庫名稱為 USERNAME.github.io，部署後網址為 https://USERNAME.github.io。
  • 專案網站：網址通常是 https://USERNAME.github.io/REPO（取決於 Pages 設定）。
  • HTTPS: GitHub Pages 自動提供憑證。
